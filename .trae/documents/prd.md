# 小提琴垂直交易与资讯咨询平台 - 产品需求文档

## 1. 产品概述

小提琴垂直交易与资讯咨询平台是一个专注于小提琴整琴、琴弓、配件、二手乐器交易的垂直电商平台，同时提供资讯导购和选琴咨询服务。平台整合了新品/二手交易、专业评测、问答咨询、验琴服务等核心功能，为小提琴爱好者、演奏者、琴行和制琴师提供一站式解决方案。

### 目标用户群体
- 小提琴初学者和家长：需要购买第一把小提琴，不懂尺寸、价格和品牌
- 小提琴学生：需要升级小提琴、琴弓或配件
- 专业演奏者：关注音色、做工、产地、制琴师和演奏表现
- 老师和培训机构：需要推荐学生用琴或批量采购
- 琴行和制琴师：希望发布新品、二手琴、寄售琴和维修服务
- 二手卖家：希望出售闲置小提琴、琴弓和配件

### 核心价值
- 专业化的小提琴交易平台，汇聚优质琴源
- 二手交易保障，平台验琴服务降低交易风险
- 专业知识导购，帮助用户做出明智购买决策
- 连接买家与卖家、老师、制琴师的交流社区

## 4. 核心业务流程

### 4.1 用户浏览和购买流程
用户打开首页 → 浏览分类或搜索 → 查看商品详情 → 联系卖家或直接购买 → 下单付款 → 等待发货 → 确认收货 → 评价

### 4.2 用户发布商品流程
注册登录 → 进入卖家中心 → 发布商品 → 填写商品信息 → 上传图片视频 → 提交审核 → 发布成功

### 4.3 咨询问答流程
用户提问 → 专家/卖家/用户回答 → 追问补充 → 问题解决或付费咨询

### 4.4 验琴服务流程
预约验琴 → 提交验琴申请 → 平台安排验琴师 → 验琴完成 → 获取验琴报告

## 6. UI/UX 设计规范

### 6.1 视觉风格
**Neo-Brutalist（新野兽派）风格**

### 6.2 色彩系统
- **主色调**：米白色 #F5F5F0（背景）
- **强调色**：
  - 黑色 #000000（边框、标题）
  - 荧光黄 #FFFF00（高亮、标签）
  - 粉红色 #FF69B4（女性相关、特殊标签）
  - 亮蓝色 #00BFFF（链接、交互元素）
- **中性色**：黑色 #000000（文字）、白色 #FFFFFF（卡片背景）
- **辅助色**：
  - 橙色 #FF6B35（促销、二手标签）
  - 绿色 #00FF00（认证通过）

### 6.3 字体系统
- **标题字体**：超粗黑体，类似 font-black，建议使用 Impact、Anton 或自定义粗体
- **正文字体**：等宽字体，类似 font-mono，建议使用 Courier New、Roboto Mono
- **字号层级**：
  - Hero 标题：72-96px
  - 页面标题：48-64px
  - 区块标题：32-40px
  - 卡片标题：20-24px
  - 正文：14-16px
  - 辅助文字：12-14px

### 6.4 边框和间距
- **边框**：黑色粗边框，3-4px
- **卡片边框**：3px solid #000
- **按钮边框**：3px solid #000
- **间距**：8px 为基础单位，16px、24px、32px、48px 递进

### 6.5 阴影效果
- **硬边缘阴影**：无模糊，4-8px 偏移，黑色
- **示例**：
  ```css
  box-shadow: 6px 6px 0px #000;
  ```
- **悬停阴影**：8px 偏移，更强效果
  ```css
  box-shadow: 8px 8px 0px #000;
  ```

### 6.6 圆角禁用
- **禁止使用圆角**：所有元素使用硬直角
- 按钮：0px border-radius
- 卡片：0px border-radius
- 输入框：0px border-radius
- 图片：0px border-radius

### 6.7 渐变禁用
- 禁止使用任何渐变效果
- 纯色背景为主

### 6.8 按钮交互
- **悬停效果**：按钮向右向下位移 4px
- **示例**：
  ```css
  .btn:hover {
    transform: translate(4px, 4px);
    box-shadow: 2px 2px 0px #000;
  }
  ```

### 6.9 卡片设计
- 白色背景
- 黑色粗边框
- 硬阴影
- 无圆角
- 清晰的商品信息展示

### 6.10 页面整体氛围
- 音乐感：字体排版有节奏感
- 年轻感：颜色鲜艳、对比强烈
- 专业感：信息组织清晰、分类明确
- 交易平台可信感：标签清晰、服务透明
